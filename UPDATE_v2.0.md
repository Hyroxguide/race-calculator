# 🎉 v2.0 업데이트 완료!

## 변경 사항

### 1. ✅ iOS Picker 스타일 휠 인터페이스 (드롭다운 → 휠)

**변경 전**: 드롭다운 선택
```
[05 ▼] : [30 ▼]
```

**변경 후**: iOS 스타일 휠 (스와이프/드래그)
```
  04
  05  ← 선택됨 (노란색 강조)
  06
```

**특징**:
- 마우스 드래그 & 터치 스와이프 지원
- 부드러운 스냅 애니메이션
- 중앙 선택 항목 노란색 강조
- iOS 네이티브 앱 느낌

---

### 2. ✅ 스테이션 입력 → 슬라이더 조정 방식

**변경 전**: 스테이션 총합 직접 입력
```
스테이션 총합: [36 ▼] : [01 ▼]
```

**변경 후**: 평균 기준 슬라이더 조정
```
빠름 (-5분) ◀━━━●━━━▶ 느림 (+5분)
           36:01 (평균)
```

**특징**:
- 평균값 자동 고정 (성별에 따라 다름)
- ±5분 범위 조정 가능 (-300초 ~ +300초)
- 실시간 시간 표시
- 색상 변화:
  - 빠름 (마이너스): 초록색
  - 평균 (0): 노란색
  - 느림 (플러스): 주황색

## 📱 사용법 (새로운 UI)

### iOS Picker 사용법

1. **마우스 사용자**: 
   - 휠 영역을 클릭 & 드래그 (위/아래)
   - 놓으면 자동으로 가까운 숫자에 스냅

2. **터치 사용자**:
   - 휠 영역을 스와이프 (위/아래)
   - iOS 네이티브 피커와 동일한 느낌

3. **빠른 조정**:
   - 빠르게 스와이프하면 여러 항목 이동
   - 천천히 드래그하면 정밀 조정

### 슬라이더 사용법

1. **드래그**: 슬라이더 핸들을 좌우로 드래그
2. **클릭**: 슬라이더 트랙 아무 곳이나 클릭
3. **실시간 반영**: 움직이는 즉시 결과 계산

**팁**: 
- 중앙 = 평균 (노란색)
- 왼쪽 = 빠름 (초록색)
- 오른쪽 = 느림 (주황색)
